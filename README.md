
## Description

Simple API consisting of:
- Authentication module
- Users module
- A postman collection


Using [Nest](https://nestjs.com/) framework & [MongosDB](https://www.mongodb.com/).

## Installation

```bash
$ npm install
```

## Running the app

Add your connection string into Config/Key{mongoURI} before run app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
